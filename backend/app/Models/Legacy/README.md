# Legacy Models

These models are from the pre-Lunar commerce system and are **DEPRECATED**.

**DO NOT use for new commerce code** - use Lunar models instead.

## Status

| Model | Status | Notes |
|-------|--------|-------|
| Product | Legacy | Use `Lunar\Models\Product` |
| Cart | Legacy | Not used in API |
| CartItem | Legacy | Not used in API |
| Order | Legacy | Use `Lunar\Models\Order` |
| OrderItem | Legacy | Use `Lunar\Models\OrderLine` |
| Discount | Legacy | Use `Lunar\Models\Discount` |

These exist for backwards compatibility only (Filament admin, migrations).
